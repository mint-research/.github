# Honours Thesis Workflow - MINT Research Group

## 1. Repository Management

* **Location:** GitHub `mint-research`.
* **Access:** You will be added as a collaborator to a private repo created for you.
* **Workflow:** Use **Draft Pull Requests** for ongoing work. Only mark as "Ready for Review" when you specifically need my feedback.

## 2. Documentation & Reports

* **OneDrive:** All documents (theses, reports, drafts) are shared via a live OneDrive link.
* **Versioning:** Work on live files; and keep local snapshots for tracking/backup.
* **Feedback:** I will use "Comments" or "Suggestions" mode. Please no attachments in the email.
